# AI Timescales

## installation

Requirements:

A HuggingFace token with acces to LLaMa3.2; request it here.

```bash
python3.9 < python3.13
```

Check the right version of pytorch for your system [here](https://pytorch.org/get-started/locally/). Then install the requirements:

```bash
pip install -r requirements.txt
```

Set the API key in `frontend/.env` and `backend/.env`.
